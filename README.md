# Business Context

Hospitals generate large volumes of operational, financial, and patient-related data every day. Transforming this information into actionable insights helps management optimize resource allocation, monitor department performance, improve financial outcomes, and support strategic decision-making.

This project simulates the work of a healthcare data analyst by converting raw hospital datasets into executive reports and an interactive management dashboard.

# Project Details

## Hospital Executive Dashboard

### Objective

Develop an interactive hospital performance dashboard to analyze financial performance, patient demographics, insurance coverage, and operational efficiency using Microsoft Excel.

### Dataset

Hospital Management System Dataset

**Tables Included**

- Admission
- Billing
- Department
- Disease / Medical Conditions
- Patient
- Ward
- Patient Insurance
- Insurance Provider

### Workflow

#### 1. Data Import

- Imported multiple CSV datasets into Power Query.

#### 2. Data Cleaning

- Removed duplicates
- Trimmed unwanted spaces
- Corrected data types
- Standardized formats
- Validated missing values

#### 3. Data Modeling

Created relationships between multiple tables using Power Pivot to build a relational data model.

#### 4. Reports Developed

- Revenue by Department
- Admissions by Department
- Top Revenue-Generating Medical Conditions
- Top Insurance Providers by Patient Count
- Gender Distribution
- Top Cities by Patient Count
- Revenue by Payment Mode
- Admission Type Distribution

#### 5. Dashboard Development

Designed an executive dashboard featuring:

- KPI Cards
- Interactive Slicers
- Pivot Charts
- Business Insights
- Executive Summary Layout
